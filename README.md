# PrintGroups-Unity
Extension for printing on the Unity console based on Groups and Subgroups. So you can control which debugs you want to see on your console<br />
<img width="796" height="286" alt="image" src="https://github.com/user-attachments/assets/635b573c-579d-4d56-b1cc-587887841246" />

# How to use:<br />
## Groups <br />
1.- Attach the "Terminal Manager" to any object of your scene, you can add an Prefix for each group and subgroups<br/>
<img width="465" height="233" alt="image" src="https://github.com/user-attachments/assets/f8173c8e-739a-4d52-9443-4afd0a165686" /><br/><br/>

2.- On the "Terminal" script, you will found the print groups that exist! Add or remove the groups you need 

<img width="344" height="177" alt="image" src="https://github.com/user-attachments/assets/5427a95e-7b93-4236-8b17-9007312ef7bc" /><br/><br/>

3.- You can use "This.Print(toPrint, group)" and it'll only be printed on the console if the groups it's listed on the Terminal Manager

<img width="439" height="76" alt="image" src="https://github.com/user-attachments/assets/6fe8b212-1465-4ad6-bb4b-c3806a32e0aa" />

## SubGroups
1.- You can create subgroups to keep groups organized<br/>
<img width="549" height="162" alt="image" src="https://github.com/user-attachments/assets/838bba9c-a44a-4bc4-9a35-deb4a35a0350" /><br/><br/>
